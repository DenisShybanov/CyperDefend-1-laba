У вкладці Network видно, що ресурси сторінки завантажуються з локального сервера 127.0.0.1, а віджет підтримки отримується з сервера 127.0.0.1:4000, який запущений через Express. Це означає, що взаємодія відбувається між різними портами одного хоста, тобто використовується cross-origin request. Доступ до ресурсів дозволяється завдяки заголовку Access-Control-Allow-Origin: *, який додається через middleware cors(). Цей заголовок дозволяє будь-якому джерелу виконувати запити до сервера.
